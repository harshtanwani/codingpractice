TAGS

  1. H Tag - Headings
  2. P Tag - Paragraph
  3. BR Tag - Break Line
  4. HR Tag - Horizontal Line
  5. I Tag - Italic Text
  6. EM Tag - Emphasis on Text(same as italic text)
  7. B Tag - Bold Text
  8. Strong Tag - Bold Text/Strong Meaning
  9. OL Tag - Ordered Lists
  10. UL Tag- Unordered Lists
  11. A Tag- Hyperlink Texts
  12. Table Tag - Helps to create a table
  13. TR Tag - Adds a table row
  14. TD Tag - Adds Table Data/Table Cell
  15. TH Tag - Table Header
  16. THead Tag - Table Heading
  17. Tbody Tag - Table Body
  18. tfoot Tag - Table Footer
  19. Label - Creates lables/texts for form fields
  20. input - Defines type, name & value for a label
  21. Enctype - defines encoding for form
